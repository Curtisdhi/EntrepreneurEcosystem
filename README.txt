To install this script, you will need to:

1. Upload all the included files to a web server.
2. Change the settings to match your own credentials in the file “js/mindmap.js”
	1. Change the Fusion table id to your own table id.
	2. Change the apikey to your Google Developer API key
3. Copy the HTML from the “index.html” file. 
   They're instructions there to tell what HTML is required.
	1. You will also have to change the api key where the Google maps script is loaded.

Depending on the desired style, you may modify the styling from the file “css/mindmap.css”
